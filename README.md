# SEZNAM OPRAVIL

## NAVODILA ZA UPORABO

---

**1. PRENESI DJANGO**

V Command Prompt (cmd) vpiši:

    pip install django

in Django se bo samodejno naložil.



**2. ZAGON STREŽNIKA**

V  Pythonu v terminal vpiši:

    python manage.py runserver

in odpri spletni naslov, ki se je izpisal. (http://127.0.0.1:8000/)
